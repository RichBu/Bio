GradeTronics - ScanTron Grading App

This app was developed by me, Rich Budek, to allow teachers to grade ScanTron forms on their own computer. No machine is needed. In addition the software prints an analytical report that lets the teacher know which questions were answered wrong and why.

Technologies used:
  • Reading JPEG file
  • Computervision to find edge of form
  • Rotating and squaring up an image
  • Scanning an image and locating bubble
  • Determining a human mark from printed one
  • Self-calibrating uniformity matrix
