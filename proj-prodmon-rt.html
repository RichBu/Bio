Production Monitoring and Messaging

During Covid, social distancing was enforced and a company needed to keep running. So, I developed this app by piecing together pieces from other apps. I had already wired the plant and networked the machines together, so it was possible to take advantage of what was there.

Even after Covid the system is used to keep the plant running 24/7 with lights out. By entering in the mobile phone number of the person on, the user or users can select which specific machines to monitor or all. When the machine shuts down, the user(s) get a text message with the machine and the fault code.

Summary:
The plant already had a real time monitoring with PLC's that I had designed earlier. By adding FREE public domain software, I was able to read the status of machines and to send text messages.

Technologies used:
Reading PLC's, storing data to real time database, make and event driven software that sends texts to a list of mobile numbers typed in.

Main screen only needs two entries. One is to enter the mobile numbers that the app should text to and then to activate it.
